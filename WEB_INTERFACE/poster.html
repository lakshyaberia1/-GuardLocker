📋 Overview

Existing honey vault systems need GPU servers, 63M+ training vaults, and deep crypto expertise — inaccessible for individuals.


GuardLocker is a locally deployable honey password vault using AES-256-GCM, deterministic decoy vault generation, honey account monitoring, and real-time Telegram alerts.


Any user can install it in under 5 minutes with 3 pip packages on any laptop — no GPU, no cloud, no expertise required.
